# 数学编程教学项目

这是一个用于数学编程教学的Jupyter项目脚手架，涵盖了从基础数学到高级应用的多个主题。

## 项目结构

```
baby_math/
├── notebooks/           # Jupyter笔记本目录
│   ├── 01_基础数学/     # 基础数学主题
│   ├── 02_线性代数/     # 线性代数
│   ├── 03_微积分/       # 微积分
│   ├── 04_概率统计/     # 概率统计
│   ├── 05_应用案例/     # 实际应用案例
│   └── 06_小学奥数/     # 小学奥数
├── utils/               # 工具函数模块
│   └── __init__.py
├── data/                # 数据文件目录
├── requirements.txt     # Python依赖包
└── README.md           # 项目说明文档
```

## 环境配置

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动Jupyter Lab

```bash
jupyter lab
```

或启动Jupyter Notebook:

```bash
jupyter notebook
```

## 学习路径

### 第一阶段：基础数学
- Python数学基础
- 数值计算入门
- 符号计算基础

### 第二阶段：线性代数
- 向量与矩阵运算
- 矩阵分解
- 特征值与特征向量

### 第三阶段：微积分
- 极限与连续
- 导数与微分
- 积分计算

### 第四阶段：概率统计
- 概率分布
- 统计推断
- 假设检验

### 第五阶段：应用案例
- 数据可视化
- 机器学习基础
- 优化问题求解

### 第六阶段：小学奥数
- 数论基础
- 组合数学
- 几何问题
- 逻辑推理

## 主要工具库

- **NumPy**: 数值计算
- **Pandas**: 数据处理
- **Matplotlib/Seaborn**: 数据可视化
- **SymPy**: 符号计算
- **SciPy**: 科学计算
- **Scikit-learn**: 机器学习

## 使用说明

1. 克隆或下载项目
2. 安装依赖包
3. 启动Jupyter Lab
4. 按照学习路径逐步学习各个主题
